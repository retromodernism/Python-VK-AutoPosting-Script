Изображения берутся из папки manual.
Скрипт постит изображение в группе вконтакте.

В конфиге нужно написать свой токен и id группы.

1 пост раз в ~30 минут (1728 секунд).
В сутки выходит 50 постов. Как раз ограничение вк (Скорость есть в конфиге).

Запуск из powershell в папке
pythonw .\script.py

Также есть автозапуск скрипта при включении компьютера:
В файлах RunAutoPosting.cmd и start.ps1 нужно изменить путь до папки скриптом на свой.
Затем добавить RunAutoPosting.cmd в автозапуск.
